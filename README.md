This project is a console-based To-Do List application built using Python.  
It allows users to add, view, and remove tasks.  
All tasks are stored in a text file so that data is saved even after the program is closed.

---

# Objective

To create a simple task management system using:

- Python Lists
- File Handling (open, read, write)
- Functions
- Loops
- Conditional Statements

---

# Technologies Used

- Python 3
- VS Code
- Terminal / Command Prompt

---

# Features

- View all tasks
- Add new tasks
- Remove tasks by number
- Stores tasks in tasks.txt
- Data persists after program restart
- Handles invalid input using try/except
